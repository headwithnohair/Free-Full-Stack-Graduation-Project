# 简历项目描述

## 项目名称：大学进阶英语学习小程序（StudyMate）

**项目时间**：2025.03 – 2025.06  
**项目角色**：前端开发（独立开发）  
**技术栈**：微信小程序原生框架 · JavaScript · WXML · WXSS · 微信云开发 · 微信同声传译插件

---

### 项目描述

独立设计并开发了一款面向大学生的英语综合学习微信小程序，涵盖单词记忆、听力听写、翻译练习、学习社区等核心功能模块，旨在构建"学-练-测-交流"一体化的移动端英语学习平台。小程序包含 **25 个页面**，与 Spring Boot 后端 RESTful API 联动。

---

### 核心工作

**1. 首页学习仪表盘与数据可视化**

- 设计学习进度看板，展示坚持天数、距考试倒计时、单词书完成百分比等核心指标
- 集成有道词典开放 API（`dict.youdao.com/infoline`），实现每日英语美句展示
- 通过 `wx.login` 获取用户 code 并调用后端接口完成微信登录态验证与用户识别

**2. 单词学习模块（背诵 + 复习双模式）**

- 实现新词背诵与旧词复习自动切换机制，基于后端返回的 `noRecite/noReview` 数据动态分配学习任务
- 集成百度翻译 TTS 接口（`fanyi.baidu.com/gettts`）实现单词英式/美式发音自动播放
- 记录用户对单词的掌握判断（认识/不认识），实时上报学习记录至后端进行遗忘曲线计算
- 实现错词本功能，学习完成后自动进入错词复习流程

**3. 听力听写训练**

- 调用后端接口随机获取待复习单词，自动播放单词发音
- 实现逐字母输入校验机制，对比用户输入与原单词，实时判断正误并反馈
- 支持查看单词释义、发音详情，错误自动记录并跳转下一词

**4. 翻译练习模块**

- 调用后端获取待翻译英文句子，用户输入中文翻译后提交并查看参考译文对照
- 引入**微信同声传译插件（WechatSI v0.3.5）**实现整句英文 TTS 朗读（`plugin.textToSpeech`）
- 点击句中单词可弹出有道词典实时查词弹窗（`dict.youdao.com/suggest`），定位计算弹窗位置
- 实现音频播放/暂停控制、300字输入限制等交互细节

**5. 学习社区（帖子广场）**

- 实现帖子列表展示、下拉刷新获取最新帖子、点击进入帖子详情页
- 使用 `eventChannel` 页面间事件通信机制传递帖子数据，实现跨页面数据共享
- 支持发帖功能页面

**6. 学习计划设置**

- 提供单词书选择、每日新词/复习量设置（双列 picker 联动选择）
- 支持考试截止日期选择、目标分数设置
- 偏好设置：英音/美音切换、自动朗读开关、字体大小调节
- 修改后通过 JSON 格式提交至后端 `setUserPlan` 接口持久化

**7. 个人中心与登录系统**

- 实现微信快捷登录（`wx.getUserProfile`）与账号密码登录双模式
- 版本兼容处理：通过 `compareVersion` 函数判断基础库版本，低版本降级使用 `getUserProfile`，高版本使用昵称头像填写能力
- 头像上传（`wx.uploadFile`）、昵称修改、用户信息管理
- 集成通知页、匿名反馈、关于我们等辅助页面

**8. 全局搜索**

- 实现单词模糊搜索功能，调用后端 `obscureSreachWord` 接口，支持输入实时查询
- 搜索结果展示单词释义并支持发音播放

---

### 技术亮点

| 亮点 | 说明 |
|------|------|
| **页面间通信** | 使用 `eventChannel` + `wx.setStorageSync` 实现复杂数据流转 |
| **Promise 封装** | 对 `wx.request` 进行 Promise 封装，支持链式调用解决回调地狱 |
| **条件渲染** | 使用 `wx:if/wx:else` 实现学习状态（背诵/复习/完成/错词）的视图动态切换 |
| **音频管理** | 使用 `wx.createInnerAudioContext` 管理单词/句子音频播放，正确处理页面卸载时的资源销毁 |
| **第三方 API 集成** | 集成有道词典、百度 TTS、微信同声传译插件等多个外部服务 |
| **自适应布局** | 动态计算胶囊按钮位置和状态栏高度，实现自定义导航栏适配 |
| **按需加载** | 配置 `lazyCodeLoading: "requiredComponents"` 优化首屏性能 |

- 完成 **25 个页面**的完整前端开发，覆盖学习全流程
- 对接 **15+ 后端 RESTful API** 接口，实现前后端数据联调
- 实现新词背诵、旧词复习、听力听写、翻译练习、社区交流 **五大功能模块**
